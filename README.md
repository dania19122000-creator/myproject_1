# Express.js Practice Project

This is a small **learning project** created to practice backend fundamentals using **Node.js and Express.js**.

The project is intentionally simple and implemented in a **single file**, focusing on understanding core concepts rather than production-ready architecture.

---

## Purpose

The main goal of this project was to:

- Understand how Express.js works under the hood
- Practice building basic REST API endpoints
- Learn request/response handling
- Work with middleware and routing
- Get familiar with Node.js backend workflow

This project was created **for educational purposes only**.

---

## Features

- Basic Express server setup
- Simple REST API endpoints
- Request handling (`req`, `res`)
- Basic middleware usage
- JSON responses

---

## Tech Stack

- Node.js
- Express.js
- JavaScript
